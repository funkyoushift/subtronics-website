# Subtronics Website Project Brain

## Core identity
Subtronics LLC should read like a trustworthy local business first:
- computer repair
- upgrades
- diagnostics
- custom PCs
- gaming PC buying guidance
- ongoing support

## Main storefront paths
- **Buy a Gaming PC** → `/gaming-pc-prices-builds/`
- **Repair / Upgrade Services** → `/services/`
- **DIY / Troubleshooting** → `/help/` and `/help/diy/`
- **Membership / retention** → `/techcare/`

## Geographic focus
Primary cities worth keeping strongest:
- Flushing
- Flint
- Grand Blanc
- Fenton
- Davison
- Burton
- Swartz Creek
- Clio
- Goodrich
- Linden

Secondary nearby coverage should only stay live if the page quality is strong enough to justify indexing.

## Page intent guardrails
- One strong gaming PC buying page beats multiple competing build pages.
- Location pages should support the main service pages, not cannibalize them.
- DIY pages should educate first, then softly convert.
- Trust pages should prove the business is real and local.

## Permanent rules
- finished website edits must come back as a deployable build
- no orphan important pages
- no duplicate intent pages without a clear reason
- no robotic filler copy
- every major structural change requires a full audit
