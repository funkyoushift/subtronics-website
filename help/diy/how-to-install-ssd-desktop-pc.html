How to Install an SSD in a Desktop PC

Adding an SSD is one of the best speed upgrades for an older desktop. Boot times, app launches, and everyday responsiveness can improve dramatically.

JayzTwoCents covers both SATA and M.2/NVMe installs, which makes this page useful whether you are upgrading an older desktop or a newer board.

Step-by-step basics

  1. Confirm whether your motherboard supports a 2.5-inch SATA SSD, an M.2 NVMe SSD, or both.
  2. Power down the PC, switch off the PSU, unplug the system, and open the case.
  3. For a 2.5-inch SATA SSD, mount the drive in the proper tray or bracket and connect both SATA data and SATA power.
  4. For an M.2 SSD, remove the slot screw, slide the drive into the slot at a slight angle, then secure it with the screw.
  5. Close the case, boot into BIOS if needed, and confirm the drive is detected.
  6. In Windows, open Disk Management, initialize the new drive, create a volume, and format it so it appears in File Explorer.
  7. If the SSD will replace your old boot drive, clone the old drive first or plan for a clean Windows install.

Things to avoid

  • Do not buy an NVMe drive for a board that only supports SATA M.2, or vice versa.
  • A brand-new drive may not appear in File Explorer until it has been initialized and formatted.
  • If the computer has important files, stop and make a backup before migrating Windows or changing the boot drive.

When to hand it off

DIY is great when the job is safe, simple, and you have time to work slowly. Once the job turns into a data-risk situation, repeated boot failures, stripped screws, BIOS confusion, or expensive-part anxiety, it usually makes sense to stop and get help.
